Description
Système de pointage et production industriel – TP Java POO
Contexte
Ce projet est réalisé dans le cadre du TP de Programmation Orientée Objet (Java) à l'ISTA Kolwezi, promotion de BAC II Informatique Industrielle.
L'application permet de gérer la présence des opérateurs et d'enregistrer les productions (pièces conformes et rebuts) via une interface console, avec stockage dans des fichiers texte/CSV.

Objectif du GROUPE 3 (Équipe A)
Notre équipe est responsable du module Menu & Navigation.
Cela inclut :

La création du menu principal interactif
La navigation entre les différentes fonctionnalités
L'appel des modules développés par les autres équipes
